`cla critic`: Ejecuta varios test de calidad en las reglas.

## Opciones

`--rule <id>/<nombre>` se ejecuta sólo para esta regla

`--type <tipo>` se ejecuta sólo para las reglas de ese tipo (por ejemplo reglas de pase)
