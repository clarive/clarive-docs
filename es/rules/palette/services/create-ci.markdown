Servicio que permite crear un nuevo **Recurso** con los atributos especificados.

Para configurar el Recurso es necesario rellenar la tabla Clave/Valor, por ejemplo:

    name    <nombre_Recurso>

Además, el usuario puede completar la tabla añadiendo más propiedades del Recurso así como los valores deseados.
